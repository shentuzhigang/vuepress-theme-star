## 使用插件

### 安装

```bash
npm i -D @starzkg/vuepress-plugin-archive
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "archive",
      {
        // 你的配置选项
      },
    ],
  ],
};
```
