## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-copy-code2
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "copy-code2",
      {
        // 你的配置选项
      },
    ],
  ],
};
```
