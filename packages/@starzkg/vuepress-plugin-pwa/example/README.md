## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-pwa2
```

### 使用

```js {7}
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "pwa2",
      {
        // your options
      },
    ],
  ],
};
```
