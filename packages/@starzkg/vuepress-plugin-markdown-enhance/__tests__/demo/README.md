### Install

```bash
npm i -D vuepress-plugin-md-enhance
```

Or

```bash
yarn add -D vuepress-plugin-md-enhance
```

### Usage

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "md-enhance",
      {
        // your options
      },
    ],
  ],
};
```
