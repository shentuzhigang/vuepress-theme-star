## 使用插件

### 安装

```bash
npm i -D vuepress-plugin-seo2
```

### 使用

```js
// .vuepress/config.js
module.exports = {
  plugins: [
    [
      "seo2",
      {
        // 你的配置选项
      },
    ],
  ],
};
```
